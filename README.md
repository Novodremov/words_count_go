## Подсчёт слов

### Комментарий

В связи с отсутствием деталей в тестовом задании после оперативного изучения синтаксиса Go написаны 2 версии программы для подсчёта частоты слов во входном тексте. Обработку возможных ошибок пока не осуществлял для сокращения времени выполнения задания.

*01_simple_solution.go* - реализованное в первую очередь простое решение: отсутствуют дополнительные параметры, текст принимается со стандартного потока ввода, регистр слов игнорируется, под словами считаются разделённые пробельными символами наборы непробельных символов, по краям очищенные от спецсимволов и знаков препинания; вывод результата осуществляется в стандартный поток вывода.

*02_complex_solution.go* - усложнённое решение (пользовался помощью ИИ, однако весь код разобран и подвергнут значительному рефакторингу), есть возможность ввода дополнительных параметров в командной строке при запуске (при отсутствии параметров в команде их значения уточняются у пользователя): выбор источника текста (файл или стандартный поток ввода), выбор учёта/неучёта регистра, выбор необходимости очистки слов от спецсимволов и знаков препинания, выбор способа вывода результата (файл или стандартный поток вывода).

### Стек:
```
- Go (Golang)
```

### Запуск проекта

Клонировать репозиторий:
```bash
git clone git@github.com:Novodremov/words_count_go.git
```

Перейти в папку с проектом:
```bash
cd words_count_go
```

Запустить решение №1:
```bash
go run 01_simple_solution.go
```

Вывод справки (возможные аргументы) для решения №2
```bash
go run 02_complex_solution.go -help
```

Пример запуска решения №2
(подсчёт слов в файле example.txt, без учёта регистра, с удалением спецсимволов, вывод результата в файл result.txt):
```bash
go run 02_complex_solution.go -file_in example.txt -file_out result.txt -case n -letters y
```
